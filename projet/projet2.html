Projet de machine learning

Dans ce projet, j'ai developpe un systeme de classification automatique d'articles de presse avec des techniques de machine learning.

Objectif technique

Concevoir un algorithme de classification supervisee capable d'identifier automatiquement le genre d'un article (sport, politique, economie, culture, etc.).

Fonctionnalites

  • Analyse du contenu textuel (titres, mots-cles, structures)
  • Classification a partir d'articles labellises
  • Evaluation avec bareme et seuils de performance

Technologies

Java et bibliotheques de traitement de texte.
