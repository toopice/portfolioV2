Projet POVE - application web

Projet universitaire de plateforme de rencontre culturelle, realise avec React pour le front-end et Node.js pour le back-end.

Objectif technique

Creer une plateforme de rencontre avec messagerie en temps reel et systeme de matching selon les preferences utilisateurs.

Fonctionnalites

  • Messagerie instantanee via WebSocket
  • Matching utilisateur
  • Exploration de lieux culturels
  • Interface React modulaire
  • Gestion des comptes et authentification

Technologies

React, Node.js, PostgreSQL, Socket.io, Express, JWT, bcrypt.
